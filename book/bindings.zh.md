
# 绑定

富有创造力的人为各种环境和编程语言编写了绑定或接口.使用其中之一可以让您利用您喜欢的语言或系统中的cURL能力.在撰写本文时,这是所有已知接口的列表.

下面列出的绑定不是curl / libcurl分发存档的一部分,但必须单独下载和安装.
